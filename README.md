## 📜 Core Rules of Engagement
To get the most out of this academic journey, I strictly adhere to the three golden rules outlined in the handbook:
1. **Type Every Line:** No copy-pasting. The learning happens in the fingers, not in the eyes.
2. **Predict Before Running:** Before executing any snippet, write down the expected output. Finding out *why* a prediction is wrong is the fastest way to master the runtime.
3. **Daily Completion:** Complete each chapter's practice exercises on the exact same day it is read.
